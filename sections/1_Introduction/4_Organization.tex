\section{Dissertation Organization}

The remainder of this dissertation is organized as follows. Chapter \ref{trust} introduces \trust{}, a delegation 
based programming abstraction that provides safe access to shared data. Chapter \ref{gossamer} presentes \name{}, 
an extension of \trust{}, that allows single-machine, multi-threaded applications built using \trust{}, to run on 
a rack consisting of many machines with minimal changes. Chapter \ref{conclusion} discusses future work and provides 
a conclusion.
